# Progress Dashboard
## Live Demo

https://tradeils1012-svg.github.io/progress-dashboard/

Мини-приложение для отслеживания финансовых целей и прогресса обучения.

## Возможности

- Расчёт финансового прогресса
- Расчёт срока достижения цели
- Отслеживание часов обучения
- Автоматический пересчёт при вводе
- Сохранение данных в браузере
- Сброс данных

## Технологии

- HTML
- CSS
- JavaScript
- LocalStorage
