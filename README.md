# Progress Dashboard

Мини-приложение для отслеживания финансовых целей и прогресса обучения.

## Live Demo

https://tradeils1012-svg.github.io/progress-dashboard/

## Возможности

### Финансовый прогресс

- Расчёт процента достижения финансовой цели
- Расчёт оставшейся суммы
- Прогноз срока достижения цели
- Выбор дедлайна
- Расчёт необходимого ежемесячного вклада
- Сравнение текущего темпа с необходимым
- Визуальный статус плана:
  - опережаешь план
  - идёшь точно по плану
  - отстаёшь от плана

### Прогресс обучения

- Цель обучения в часах
- Учёт пройденных часов
- Расчёт процента прогресса
- Автоматический пересчёт

### Работа с данными

- Сохранение данных в LocalStorage
- Восстановление данных после перезагрузки
- Полный сброс данных
- Автоматический пересчёт при изменении значений
