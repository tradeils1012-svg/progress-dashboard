# Money Goal Tracker

Это первое минимальное приложение для обучения.

## Как запустить на MacBook

1. Распакуй папку.
2. Открой файл `index.html` двойным кликом.
3. Введи цель и текущую сумму.
4. Нажми `Посчитать`.

## Какие навыки ты тут изучаешь

HTML  = тело приложения
CSS   = внешний вид
JS    = мозг и действия
